# gulp-postcss
移动端自适应VW布局-----工作流构建


## 使用方法

#### 1.克隆项目
```console
$ git clone https://github.com/dc233/gulp-static-moblie.git
```

#### 2.安装依赖
```console
$ cd gulp-static-moblie && npm  install
```

#### 3.项目运行
```console
$ gulp
```
